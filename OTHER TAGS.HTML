OTHER TAGS

LINE BREAK

Line break helps to
break a line. this tag is a very unique tag as it opens and never closes.

HORIZONTAL RULE

Horizontal rule tag helps us to sketch a horizontal line between paragraphs.

PREFORMATTED TEXT

Preformatted text, often enclosed within pre tags in HTML, refers to text that 

retains its original formatting, including spaces, line breaks, and tabs, when displayed on a webpage.

PARAGRAPH TAG

The paragraph tag is used to show a beginning of a paragraph.
